# IPL Cricket Data Analysis

## Project Overview
This project analyzes IPL match data to find insights such as:
- Most successful teams
- Impact of toss on match results
- Number of matches played per season

## Dataset
- matches.csv (IPL match-level data)

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Google Colab

## Key Insights
- Some teams consistently perform better across seasons
- Winning the toss does not guarantee winning the match
- IPL has grown significantly over the years
